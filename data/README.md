# Dataset

The ASL Alphabet dataset is obtained from Kaggle:
https://www.kaggle.com/datasets/grassknoted/asl-alphabet

Due to size and licensing constraints, the full dataset is not included.

To reproduce the experiments:
1. Download the dataset from Kaggle
2. Place it in `data/raw/`
3. Run `preprocess_asl.py`
