This directory contains the **American Sign Language (ASL) alphabet dataset** used for training and evaluating our real-time ASL recognition model.

Due to size constraints, the full dataset is **not stored directly in this repository**. Instead, this folder documents the expected dataset structure and organization.

---

## Dataset Source

**Kaggle ASL Alphabet Dataset**  
https://www.kaggle.com/datasets/grassknoted/asl-alphabet

The dataset consists of labeled RGB images of hand gestures corresponding to ASL alphabet letters.

---

## Directory Structure

Each subfolder corresponds to a single ASL letter class and contains images of that hand gesture.
